## Käyttöohje

### Pelaaminen AI:ta Vastaan

Tekoälyä vastaan voi pelata [TIRA AI-platform alustalla](https://github.com/game-ai-platform-team/tira-ai-local).

### Asennus ja Käyttö

1. **Kloonaa repositorio:**  
 `git clone https://github.com/JoonaPietarinen/shakki-tekoaly`


2. **Käynnistä AI-platform:**  
 Seuraa AI-platformin [ohjeita](https://algolabra-hy.github.io/aiplatform-fi#:~:text=Minimaaliset%20ohjeet%20k%C3%A4ytt%C3%B6%C3%B6nottoon)


3. **Pelaa:**  
- Valitse peliksi shakki
- Raahaa tämän projektin juurikansio AI-platformin "Submit your AI" -osioon
- Tee siirtoja shakkilaudalla
- Tekoäly vastaa omalla siirrolla
- (Ei kannata koskea alustan nuolinäppäimiin tai vastustajan pelinappuloihin, voi sekottaa AI:n)
