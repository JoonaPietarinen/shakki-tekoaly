# Viikkoraportti 1
Osallistuin aloitusluentoon, mitä ennen perehdyin kurssin ohjeisiin ja mahdollisiin aiheisiin. Valitsin aiheeksi pelitekoälyn ja spesifisesti shakki-tekoälyn omien mielenkiintojeni perusteella. Perehdyin aiheeseen ja valitsin siihen sopivat algoritmit (negamax + alpha-beta + transpositiotaulu). Lopuksi suoritin aloitustoimenpitee (mm. github, labtool, määrittelydokumentti).

**Mitä opin tällä viikolla?**  
Opin miten kurssin tekoälyalusta toimii shakkia varten.

**Mikä jäi epäselväksi tai tuotti vaikeuksia?**  
Suurin pohdinta liittyi aluksi siihen, mikä taso algoritmeissa riittää kurssille ja missä vaiheessa aiheesta tulee liian laaja
